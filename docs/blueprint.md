# **App Name**: Invoice Overview

## Core Features:

- Invoice List: Display a list of invoices fetched from a provided endpoint using GET requests.
- Invoice Status: Indicate the validation status of each invoice with visual cues: green for validated, and a prompt to send to review for unvalidated ones.
- Dynamic Notifications: Employ an AI-powered tool to monitor the invoices and to suggest any important upcoming actions based on status.

## Style Guidelines:

- Primary color: Deep blue (#00377B), mirroring the reference image, for a professional and trustworthy feel.
- Background color: Light desaturated blue (#E5EAF2) for a clean, corporate background.
- Accent color: Bright cyan (#00A9E0), for call-to-action buttons, links, and highlights.
- Font: 'Inter' sans-serif font for body and headline, for a modern, neutral appearance.
- Cards used to represent invoices; clear sections for validated/unvalidated invoices.
- Simple, flat icons for invoice status, download options, and other actions.
